# 📚 2tab — Second-Hand Book Marketplace

2tab is a second-hand book marketplace built for people in Azerbaijan.
It provides a simple and centralized platform to buy and sell used books at affordable prices.

---

## 🚀 Problem

People often struggle to find affordable books, and there is no easy, centralized place to buy or sell second-hand books.

---

## 💡 Solution

2tab solves this problem by allowing users to:

* Browse available books with details
* Upload and sell their own books
* Add books to favorites
* Manage listings via a personal dashboard

---

## ✨ Features

* 📖 Book listing with price, condition, and seller info
* ❤️ Add / remove favorites
* 👤 User authentication system
* 📊 Personal dashboard (manage books)
* 🖼 Secure image upload system
* 📱 Mobile-friendly UI

---

## 🔐 Security Features

* Prepared statements (SQL Injection protection)
* CSRF protection
* Output escaping (XSS prevention)
* Session security (httponly, samesite, regeneration)
* File upload validation & isolation
* Basic security headers

---

## 🛠 Tech Stack

* PHP (Vanilla)
* MySQL
* HTML / CSS / JavaScript

---

## ⚙️ Installation

```bash
git clone https://github.com/ResulMemmedzade/2tab-marketplace.git
cd 2tab-marketplace
```

Create `.env` file based on `.env.example`

```env
APP_ENV=local
APP_DEBUG=true
APP_BASE_URL=/2tab

DB_HOST=localhost
DB_NAME=2tab
DB_USER=root
DB_PASS=
```

Run on local server (e.g. WAMP / XAMPP)
